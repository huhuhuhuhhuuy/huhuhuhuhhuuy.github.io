      原则就是，在每次换位置的时候，将大的换到右边，直到卡住，所以每次起码能解决将当前最大的排到正确的位置上
      也就是说交换复杂度最多为O^ 2
      1）两层循环必不可少
      考虑提前结束冒泡，已排好的情况，当一次冒泡中不再需要交换两元素位置时，说明排序结束
      2）设置标志位判断循环是否已经结束
      3）记住最后一次交换的位置，后面的直接不用走判断流程了，都是已经排好了的

    交换函数的技巧：

    • 普通的交换函数
      ‘
      swap(a,b){
      let temp = a;
      a = b;
      b = temp;
      }
      ’
    • 不引入新变量进行
      ‘
      swap1(a,b){
      a = a + b;
      b = a - b;
      a = a - b;
      }
      ’
      可能存在的问题是数字越界
      ‘
      a = a ^ b
      b = a ^ b
      a = a ^ b
      ’
